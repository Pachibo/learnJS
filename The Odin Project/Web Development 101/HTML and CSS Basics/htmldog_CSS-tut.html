px is for pixels. Means size at 100%
em is for font size multiplier. (ie. 3em is three times the font size)
pt is for points, the unit typically used in printed media
% is for percentage
there are also pc (picas), cm, mm, and inches
no need to specify units for zero

CSS uses RGB colors in the form of

  • color name (very limited, ie. red)
  • rgb(int,int,int), where int is 0-255 inclusive
  • rgb(%,%,%)
  • #hexadecimal with six or three digits that also correspond to rgb
Can even define things like hue, saturation, transparency. Google for tnat stuff.

can define font-family (ie. Arial and Times New Roman)
can import custom fonts as well
can manipulate font-size
can also bold things with font-weight. Google for all ways to manipulate font.
can use text-decoration to underline, overline, etc.

margin and padding are two of the most used properties for spacing out elements.
margin is for space outside something
padding is for space inside something
These properties are based on the following box model.

Margin box
Border box
Padding box
Element box

There different border-style, such as solid, dotted, ridged, etc.
There's also border-width and border-color

there are class selectors, which use ".", and ID selectors, which use "#", in CSS.
an ID is for only one element, while a class can be used for multiple elements.
Can also select specific elements with a specific class.
For example, p.good would only be applied to paragraph elements with the class "good"

For elements/ID/class that you want to have the same properties, can just separate by commas
For example, h1, .class1, #ID1 {properties}

properties can also be nested.
for example, if you want all #bad to have "background-color: orange" and all h2 with #bad to have "color:gray", you can do this:
#bad { background-color: orange }
#bad h2 {color:orange}

properties can also be applied to pseudoclasses in the format of selector:pseudo_class {prop}
For example, a:link and a:visited
there are also dynamic pseudo classes such as
active: when something is activated (like clicking)
hover: when something is passed over by an input (like cursor moving over)
focus: when something is selected (such as selecting by keyboard)

The first-child pseudo class is literally what it means.
It styles the first child, such as the first paragraph of a body

There are short-hand ways to designate properties.
For example, the margin-top to margin-left can just be
margin: top right bottom left (clockwise)
it's the same for borders. border: width color style
also for font: style weight size height family

Background images can be made with
background: color image repeat position
image is in the form of url(image url)

In CSS more specificity means greater precedence

There are a few display types:
inline: displayed in a line
block: each box takes up a line
inline-block: a more customizable inline
there are also display types for tables. Google them if needed
use list-item to display boxes in a list, nested within ul or ol.

use selector::pseudo_element to designate
examples are first-line and first-letter
can also designate before and after
coupled with the content property, which can be:
open-quote, close-quote, string in quotes, or image

position property can be applied to a box
static: default value, where it's in normal order
relative: static, but can designate top, right, bottom, left
absolute: can be placed in absolute positions of the page (same 4 positions)
fixed: absolute, but with respect to browser window.
